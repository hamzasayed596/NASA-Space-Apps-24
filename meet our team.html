Project Description

Orrery Web App for Near-Earth Objects

Our project aims to develop an interactive web app that visualizes the orbits of near-Earth objects in real time.

Key Features:

  • Visualization of NEO orbits and their proximity to Earth.
  • Access to detailed information about NEOs, such as size, distance, and trajectory.
  • A 3D orrery model, utilizing web technologies for an immersive user experience.
  • Integration of real-time data from NASA’s API and other astronomical datasets to ensure accuracy and relevance.

Our Vision

We aspire to create a platform that educates the public about NEOs, raises awareness about space hazards, and inspires interest in space exploration.

Our Goal

00100-Keplers’ goal is to develop a user-friendly, visually appealing orrery web app that serves as both an educational tool and a resource for exploring near-Earth objects.
